# 🛒 E-Commerce Platform Exploratory & Functional Test Suite

Conducted comprehensive manual testing on the Sauce Demo e-commerce platform using the `problem_user` credentials. This project demonstrates my ability to analyze requirements, write clear step-by-step test cases, track system logic failures using Chrome DevTools, and author reproducible bug reports.

## 📁 Project Deliverables

* **[TEST_CASES.md](TEST_CASES.md):** The structured, step-by-step test scripts used to validate the application's core features against expected behavior.
* **[BUG_REPORT_LOG.md](BUG_REPORT_LOG.md):** A technical log detailing the functional defects found during testing, including replication steps and severity levels.
* **Evidence Captures:** Visual screenshots capturing the application failures alongside underlying browser console errors (`Bug 1.png`, `Bug 2.png`, `Bug 3.png`).

## 🛠️ Technologies & Tools Used

* **Testing Type:** Manual Functional Testing, Black-box Testing, Exploratory Testing
* **Environment:** Google Chrome (Desktop)
* **Defect Tracking:** Markdown Bug Logging (Simulating Jira issue creation)
* **Investigation Tools:** Chrome DevTools (Console log analysis & network monitoring)

## 🎯 Core Features Tested

1. **Inventory Page:** Verified that product images dynamically load and match their listed details.
2. **Product Routing:** Checked that clicking individual items correctly takes the user to the specific item details and pricing page.
3. **Checkout Workflow:** Validated form fields (First Name, Last Name, Zip Code) for character retention and proper submission routing.
